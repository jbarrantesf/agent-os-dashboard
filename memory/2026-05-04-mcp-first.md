## 🔄 Arquitectura MCP Bidireccional

```
HERMES Session                    ORBIT Session
(ID: hermes-taskworker-...)      (ID: orbit-worker-...)
     │                                 │
     ├─ sessions_send() ────────────►  ├─ Receive
     │                                 ├─ Process
     │                                 └─ Execute
     │                                 │
     ◄──────────── sessions_send() ────┤
     ├─ Receive result
     ├─ Process
     └─ Send Telegram (milestone)

Zero Supabase (por ahora)
Zero polling loops (push-based)
Directo: request → execute → response
```
